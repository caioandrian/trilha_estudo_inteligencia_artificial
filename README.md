# Trilha de Estudos — Inteligência Artificial

Trilha de estudos gratuita e interativa sobre Inteligência Artificial, organizada em 4 áreas de conteúdo com 10 tópicos.

🔗 **Acesse a trilha:** [caioandrian.github.io/trilha_estudo_inteligencia_artificial](https://caioandrian.github.io/trilha_estudo_inteligencia_artificial/)

---

## Conteúdos

| Área | Tópicos |
|---|---|
| Transformação Digital e IA | Fase 1: Digitalização, Fase 2: Extração de Dados |
| IA Clássica | IA como Máquina Preditiva, Ciclo Virtuoso da IA, IA e Tomada de Decisão, Canvas da IA Clássica |
| IA Generativa | Capacidades da LLM, Canvas da IA Generativa |
| Prompt de Alta Performance | Prompt de Alta Performance |

## Como usar

- Selecione uma área no menu de conteúdos
- Clique em um tópico na barra lateral para abrir o conteúdo
- Leia o material e clique em **Marcar como lido** para registrar o progresso
- O progresso é salvo automaticamente no navegador
- Nos tópicos de Canvas, acesse o **Modelo Interativo** para estruturar e salvar seus projetos

## Tecnologias

Site estático — HTML, CSS e JavaScript puro, sem dependências ou build. Deploy via GitHub Actions no GitHub Pages.
